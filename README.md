Esta aplicación Web esta diseñada para Búsqueda de usuarios de GitHub, por su respectivo nombre de usuario

En este se muestra el Nombre de Usuario, Avatar , Url del perfil, Compañía y no tiene tiene compañía se señala como "no afiliado" y el numero de repositorios

Tiene característica de cambio de color del tema de Claro a Oscuro y viceversa.

Requisititos 

Navegador Web como (Chrome, Firefox, Opera , Brave ,...)

Instruciones para su ejecución:

Descargar desde GitHub :  https://github.com/aandyop/Actividad_4.1/tree/main

Abrir el VSC y abrir la carpeta 

Instalar extensiones necesarias (Virtual Server)

Después de la Activación se abrirar la pagina lista para ser usada

¿Como usarlo?

Escribe el nombre de usuario de GitHub que deseas encontrar en el campo de búsqueda "¿Qué usuario de GitHub estás buscando?" y se mostraran los resultados , en este caso 3 usuarios mas acertados a tu búsqueda, y se mostrara la imagen de avatar , usuario, repos y si tiene empresa
Puedes ver los perfiles de los usuarios a detalles haciendo click en el botón de "Ver perfil de usuario" te redirecciona a la pagina de GitHub

Tema:
Puedes cambiarlo en el botón en la esquina superior derecha
